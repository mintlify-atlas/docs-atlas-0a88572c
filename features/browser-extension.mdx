## Overview

The Vega AI browser extension (coming soon) will enable you to capture job postings directly from job boards and company career pages with a single click. Save jobs to your Vega AI dashboard without manual copying and pasting.

</Note>

## How It Will Work

<Steps>
  <Step title="Install Extension">
    Install the Vega AI extension from the Chrome Web Store or Firefox Add-ons.
  </Step>
  
  <Step title="Sign In">
    Sign in with your Vega AI account credentials.
  </Step>
  
  <Step title="Browse Job Boards">
    Visit LinkedIn, Indeed, or any other supported job platform.
  </Step>
  
  <Step title="Click Extension Icon">
    When viewing a job posting, click the Vega AI extension icon.
  </Step>
  
  <Step title="Review Details">
    The extension will auto-fill job details. Review and edit if needed.
  </Step>
  
  <Step title="Save to Dashboard">
    Click "Save" to add the job to your Vega AI dashboard.
  </Step>
  
  <Step title="Access in Vega AI">
    The job immediately appears in your dashboard, ready for tracking and AI analysis.
  </Step>
</Steps>

## API for Custom Integrations

Developers can build custom integrations using the Vega AI API:

```go
// Job creation endpoint in Vega AI
POST /api/jobs/new

// Request handler
func (h *JobHandler) CreateJob(c *gin.Context) {
    // Extract form data
    title := strings.TrimSpace(c.PostForm("title"))
    description := strings.TrimSpace(c.PostForm("description"))
    companyName := strings.TrimSpace(c.PostForm("company_name"))
    
    // Create job with options
    job, isNew, err := h.service.CreateJob(
        c.Request.Context(),
        userID,
        title,
        description,
        companyName,
        options...,
    )
    
    // Return response
    if isNew {
        c.JSON(http.StatusCreated, gin.H{
            "success": true,
            "job": job,
            "message": "Job added successfully!",
        })
    }
}
```
